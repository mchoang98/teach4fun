# Bài 01: Bài tập Làm quen với Python

## Dạng đề lập trình thi đấu

## Kiến thức sử dụng

Trong các bài tập này, học sinh chỉ cần dùng:

```python
print()
```

Có thể dùng thêm ký tự xuống dòng:

```python
\n
```

Chưa sử dụng:

* Biến
* `input()`
* Kiểu dữ liệu
* Phép tính
* Điều kiện
* Vòng lặp

# Bài 7 nâng cao: Vẽ ngôi nhà bằng ký tự

## Đề bài

Viết chương trình in ra hình ngôi nhà bằng các ký tự đặc biệt.

Bài này yêu cầu học sinh chú ý:

* Khoảng trắng ở đầu dòng
* Ký tự `/`
* Ký tự `\`
* Cách in dấu `\` trong Python

## Input

Không có dữ liệu vào.

## Output

In ra hình ngôi nhà đúng mẫu.

## Ví dụ

Thay vì hiển thị trong bảng (dễ gây lỗi giao diện), hình mẫu được trình bày như sau:

**Output**

```text
    /\
   /  \
  /____\
  |    |
  | [] |
  |____|
```

## Gợi ý

Trong Python, nếu muốn in dấu `\`, ta cần viết là `\\`.

```python
print("    /\\")
print("   /  \\")
print("  /____\\")
print("  |    |")
print("  | [] |")
print("  |____|")
```
