# Thực hành bài 11-1: Đọc và ghi file

## Bài 1: Ghi mục tiêu học tập

### Mô tả

Nhập `n` mục tiêu học tập và ghi vào file `goals.txt`, mỗi mục tiêu một dòng. Sau đó đọc lại file và in ra màn hình.

### Input

* Dòng 1: số mục tiêu `n`.
* `n` dòng tiếp theo: nội dung mục tiêu.

### Output

In lại các mục tiêu, mỗi mục tiêu một dòng theo đúng thứ tự nhập.

### Ràng buộc

* `1 <= n <= 20`
* Mỗi mục tiêu dài không quá 100 ký tự.

### Ví dụ

Input:

```text
2
Hoc ham
Lam bai tap file
```

Output:

```text
Hoc ham
Lam bai tap file
```

## Bài 2: Đếm dòng nhật ký

### Mô tả

Cho file `diary.txt` đã có sẵn nhiều dòng. Viết chương trình đọc file và in số dòng không rỗng.

### Input

Không có input từ bàn phím.

### Output

```text
So dong: <count>
```

### Ràng buộc

* Bỏ qua dòng chỉ chứa khoảng trắng.
