# Thực hành bài 8-1: Danh sách

## Bài 1: Giỏ quà lớp học

### Mô tả

Nhập danh sách món quà. Sau đó nhập một món cần kiểm tra. Nếu có trong danh sách, in `FOUND`, ngược lại in `MISSING`.

### Input

* Dòng 1: số món `n`.
* Dòng 2: `n` món quà.
* Dòng 3: món cần kiểm tra.

### Output

`FOUND` hoặc `MISSING`.

### Ràng buộc

* `1 <= n <= 100`
* Tên món không chứa khoảng trắng.

### Ví dụ

Input:

```text
4
but sach keo vo
keo
```

Output:

```text
FOUND
```
