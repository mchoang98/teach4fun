# Thực hành bài 8-3: Set và Tuple

## Bài 1: Vé tham gia không trùng

### Mô tả

Nhập danh sách mã vé có thể bị trùng. Hãy in số lượng mã vé duy nhất.

### Input

* Dòng 1: số mã vé `n`.
* Dòng 2: `n` mã vé.

### Output

```text
So ve hop le: <count>
```

### Ràng buộc

* `1 <= n <= 200`
* Mã vé không chứa khoảng trắng.

### Ví dụ

Input:

```text
6
A01 A02 A01 B03 A02 C04
```

Output:

```text
So ve hop le: 4
```
