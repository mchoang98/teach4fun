# Thực hành bài 8-2: Từ điển

## Bài 1: Mã màu đội nhóm

### Mô tả

Nhập danh sách đội và màu áo. Sau đó nhập tên đội cần tra cứu màu.

### Input

* Dòng 1: số đội `n`.
* `n` dòng tiếp theo: `<doi> <mau>`.
* Dòng cuối: tên đội cần tìm.

### Output

Màu áo của đội hoặc `NOT FOUND`.

### Ràng buộc

* `1 <= n <= 50`
* Tên đội và màu không chứa khoảng trắng.

### Ví dụ

Input:

```text
3
Alpha red
Beta blue
Gamma green
Beta
```

Output:

```text
blue
```
