# Thực hành bài 6-2: Phương thức xử lý chuỗi

## Bài 1: Làm sạch tên đăng nhập

### Mô tả

Nhập một tên đăng nhập có thể thừa khoảng trắng và chữ hoa. Hãy bỏ khoảng trắng hai đầu và chuyển về chữ thường.

### Input

* Một dòng tên đăng nhập.

### Output

Tên đăng nhập sau khi làm sạch.

### Ràng buộc

* Chuỗi dài từ 1 đến 100 ký tự.

### Ví dụ

Input:

```text
   CodeMaster   
```

Output:

```text
codemaster
```

## Bài 2: Tạo hashtag lớp học

### Mô tả

Nhập tên chủ đề gồm nhiều từ. Hãy tạo hashtag bằng cách bỏ khoảng trắng hai đầu, viết hoa chữ cái đầu mỗi từ, xóa khoảng trắng giữa các từ và thêm dấu `#` ở đầu.

### Input

* Một dòng chủ đề.

### Output

Hashtag sau xử lý.

### Ràng buộc

* Chủ đề dài từ 1 đến 100 ký tự.

### Ví dụ

Input:

```text
  hoc python vui  
```

Output:

```text
#HocPythonVui
```
