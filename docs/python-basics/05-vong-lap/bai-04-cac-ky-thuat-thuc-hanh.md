# Thực hành bài 5-4: Các kỹ thuật lặp

## Bài 1: Xếp hạng điểm bằng enumerate

### Mô tả

Nhập danh sách điểm đã được sắp theo thứ hạng. Dùng `enumerate()` để in thứ hạng và điểm.

### Input

* Dòng 1: số điểm `n`.
* Dòng 2: `n` điểm.

### Output

Mỗi dòng: `Hang <i>: <diem>`.

### Ràng buộc

* `1 <= n <= 100`

### Ví dụ

Input:

```text
3
9 8.5 8
```

Output:

```text
Hang 1: 9
Hang 2: 8.5
Hang 3: 8
```
