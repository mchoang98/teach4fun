# Thực hành bài 5-1: Vòng lặp for

## Bài 1: Điểm danh nhanh

### Mô tả

Nhập danh sách tên học sinh và in lời chào cho từng bạn.

### Input

* Dòng 1: số học sinh `n`.
* Dòng 2: `n` tên.

### Output

Mỗi dòng theo mẫu `Xin chao <ten>`.

### Ràng buộc

* `1 <= n <= 50`
* Tên không chứa khoảng trắng.

### Ví dụ

Input:

```text
3
An Binh Chi
```

Output:

```text
Xin chao An
Xin chao Binh
Xin chao Chi
```
