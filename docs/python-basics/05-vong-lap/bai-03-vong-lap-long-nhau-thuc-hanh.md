# Thực hành bài 5-3: Vòng lặp lồng nhau

## Bài 1: Bảng ghế lớp học

### Mô tả

Nhập số hàng và số cột. In sơ đồ ghế dạng `R<hang>C<cot>`.

### Input

* Dòng 1: số hàng.
* Dòng 2: số cột.

### Output

Mỗi hàng là một dòng ghế, các ghế cách nhau bởi dấu cách.

### Ràng buộc

* `1 <= hang, cot <= 20`

### Ví dụ

Input:

```text
2
3
```

Output:

```text
R1C1 R1C2 R1C3
R2C1 R2C2 R2C3
```
