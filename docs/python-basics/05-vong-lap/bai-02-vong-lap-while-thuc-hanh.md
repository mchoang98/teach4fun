# Thực hành bài 5-2: Vòng lặp while

## Bài 1: Đếm ngược khai giảng

### Mô tả

Nhập số `n`, in các số từ `n` về 1, sau đó in `START`.

### Input

* Một số nguyên `n`.

### Output

Các số đếm ngược, mỗi số một dòng, cuối cùng là `START`.

### Ràng buộc

* `1 <= n <= 100`
* Nên dùng `while`.

### Ví dụ

Input:

```text
3
```

Output:

```text
3
2
1
START
```
