# Thực hành bài 3-5: Toán tử bitwise

## Bài 1: Bật quyền truy cập

### Mô tả

Một người dùng có mã quyền ban đầu `mask`. Quyền mới cần bật nằm ở bit thứ `k`, tính từ 0. Hãy in mã quyền sau khi bật bit đó.

### Input

* Dòng 1: số nguyên `mask`.
* Dòng 2: vị trí bit `k`.

### Output

Mã quyền mới.

### Ràng buộc

* `0 <= mask <= 100000`
* `0 <= k <= 20`

### Ví dụ

Input:

```text
8
1
```

Output:

```text
10
```
