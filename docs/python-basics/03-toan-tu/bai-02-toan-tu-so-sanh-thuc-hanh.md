# Thực hành bài 3-2: Toán tử so sánh

## Bài 1: Kỷ lục chạy có tốt hơn không?

### Mô tả

Trong chạy bộ, thời gian càng nhỏ thì kết quả càng tốt. Nhập thời gian chạy cũ và thời gian chạy mới, sau đó in kết quả kiểm tra thời gian mới có tốt hơn thời gian cũ không.

### Input

* Dòng 1: thời gian cũ, số thực.
* Dòng 2: thời gian mới, số thực.

### Output

In `True` nếu thời gian mới nhỏ hơn thời gian cũ, ngược lại in `False`.

### Ràng buộc

* Thời gian là số thực dương, tính bằng giây.
* Chưa dùng `if`, chỉ in biểu thức so sánh.

### Ví dụ

Input:

```text
12.5
11.8
```

Output:

```text
True
```

## Bài 2: Điểm có đạt không?

### Mô tả

Một bài kiểm tra cần từ 5 điểm trở lên để đạt. Hãy in kết quả kiểm tra điểm có đạt hay không.

### Input

* Một số thực là điểm bài kiểm tra.

### Output

`True` nếu điểm từ 5 trở lên, ngược lại `False`.

### Ràng buộc

* `0 <= diem <= 10`

### Ví dụ

Input:

```text
4.75
```

Output:

```text
False
```
