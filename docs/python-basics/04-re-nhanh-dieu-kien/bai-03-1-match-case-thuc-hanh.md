# Thực hành bài 4-3.1: Match case cơ bản

## Bài 1: Lệnh robot lớp học

### Mô tả

Robot nhận lệnh `W`, `A`, `S`, `D`. In hướng di chuyển tương ứng: `UP`, `LEFT`, `DOWN`, `RIGHT`. Lệnh khác in `UNKNOWN`.

### Input

* Một ký tự lệnh.

### Output

Hướng di chuyển hoặc `UNKNOWN`.

### Ràng buộc

* Nên dùng `match case`.

### Ví dụ

Input:

```text
A
```

Output:

```text
LEFT
```
