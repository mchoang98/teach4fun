# Thực hành bài 4-1: If else

## Bài 1: Chọn chế độ học

### Mô tả

Dựa vào số phút còn lại trong ngày, chương trình gợi ý chế độ học. Từ 60 phút trở lên in `DEEP WORK`, từ 25 đến 59 in `QUICK PRACTICE`, dưới 25 in `REVIEW`.

### Input

* Một số nguyên là số phút còn lại.

### Output

Một trong ba gợi ý trên.

### Ràng buộc

* `0 <= minutes <= 1440`

### Ví dụ

Input:

```text
40
```

Output:

```text
QUICK PRACTICE
```
