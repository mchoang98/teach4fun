# Thực hành bài 12-2: PEP 8

## Bài 1: Sửa khoảng trắng

### Mô tả

Viết chương trình tính diện tích hình chữ nhật theo PEP 8: tên biến rõ nghĩa, có khoảng trắng quanh toán tử, không viết nhiều lệnh trên một dòng.

### Input

* Dòng 1: chiều dài.
* Dòng 2: chiều rộng.

### Output

```text
Dien tich: <area>
```

### Ràng buộc

* `0 <= chieu_dai, chieu_rong <= 100000`

### Ví dụ

Input:

```text
5
3
```

Output:

```text
Dien tich: 15
```

## Bài 2: Import đúng chuẩn

### Mô tả

Dùng module `math` để tính chu vi đường tròn. Code cần đặt import ở đầu file và tên biến theo `snake_case`.

### Input

* Một số thực là bán kính.

### Output

Chu vi làm tròn 2 chữ số.

### Ràng buộc

* `0 <= ban_kinh <= 10000`

### Ví dụ

Input:

```text
2
```

Output:

```text
12.57
```
