## Mục tiêu chương

Sau chương này, học viên cần:

- Hiểu các lựa chọn compute quan trọng trong AWS.
- Nắm chắc Lambda vì đây là service trọng tâm của AWS Developer Associate.
- Biết EC2 ở mức developer cần dùng.
- Biết Elastic Beanstalk phù hợp khi deploy web app nhanh.
- Làm được câu hỏi trắc nghiệm tình huống về Lambda, EC2 và Elastic Beanstalk.

## Cách học đề xuất

- Học từng bài theo thứ tự.
- Sau mỗi bài làm lab nhỏ nếu có AWS account.
- Ghi lại keyword tiếng Anh.
- Cuối chương làm bài kiểm tra 30 câu.
- Nếu dưới 70%, học lại các bài bị sai theo phần gợi ý ôn tập.
