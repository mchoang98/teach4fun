# Chương 1. AWS Foundation cho Developer

## Mục tiêu chương

Sau chương này, học viên cần:

- Hiểu AWS là gì và vì sao developer cần học AWS.
- Biết các khái niệm nền tảng: Region, Availability Zone, Account, Root user, IAM.
- Biết dùng AWS Console ở mức cơ bản.
- Biết cài và cấu hình AWS CLI.
- Hiểu AWS SDK và programmatic access.
- Hiểu Shared Responsibility Model.
- Nhận diện được keyword thường gặp trong đề thi AWS Developer Associate.

## Gợi ý cách học

Mỗi bài nên học theo thứ tự:

1. Đọc mục tiêu.
2. Đọc phần giải thích dễ hiểu.
3. Ghi lại keyword tiếng Anh.
4. Làm lab nhỏ nếu có tài khoản AWS.
5. Làm câu hỏi ôn tập.
6. Cuối chương làm bài kiểm tra trắc nghiệm.

## Kết quả cần đạt cuối chương

- Biết AWS account hoạt động như thế nào.
- Không dùng Root user cho công việc hằng ngày.
- Biết bật MFA để bảo vệ tài khoản.
- Biết tạo IAM user/role ở mức cơ bản.
- Biết AWS CLI và AWS SDK dùng để làm gì.
- Hiểu Region và Availability Zone.
- Hiểu trách nhiệm nào thuộc AWS, trách nhiệm nào thuộc người dùng.
