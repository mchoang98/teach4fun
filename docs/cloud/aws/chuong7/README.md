# Chương 7. Security nâng cao cho Developer

## Mục tiêu chương

Sau chương này, học viên cần:

- Hiểu các khái niệm bảo mật nâng cao thường gặp trong AWS Developer Associate.
- Biết dùng Cognito để xác thực người dùng cho web/mobile app.
- Biết phân biệt Cognito User Pool và Cognito Identity Pool.
- Biết dùng KMS để mã hóa dữ liệu.
- Biết dùng Secrets Manager và Parameter Store để lưu secret/config an toàn.
- Biết áp dụng security best practices cho ứng dụng cloud-native.
- Nhận diện được keyword tiếng Anh trong đề thi DVA-C02.
