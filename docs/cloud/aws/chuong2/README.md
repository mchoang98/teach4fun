# Chương 2. IAM và Security nền tảng

Chương này tập trung vào IAM và các nguyên tắc bảo mật nền tảng trong AWS Certified Developer - Associate DVA-C02.

## Danh sách bài học

1. Bài 1. IAM User, Group, Role và Policy
2. Bài 2. IAM Policy JSON
3. Bài 3. Least Privilege
4. Bài 4. IAM Role cho Lambda và EC2
5. Bài 5. STS và AssumeRole
6. Bài 6. IAM Policy Evaluation
7. Bài kiểm tra cuối chương 2

## Mục tiêu chương

Sau chương này, học viên cần:

- Phân biệt được IAM User, Group, Role và Policy.
- Đọc hiểu được IAM Policy JSON cơ bản.
- Biết áp dụng nguyên tắc Least Privilege.
- Biết dùng IAM Role cho Lambda và EC2.
- Hiểu Temporary Credentials, STS và AssumeRole.
- Hiểu cách AWS đánh giá quyền Allow/Deny.
- Nhận diện được keyword thường gặp trong đề thi DVA-C02.
