# Dự án cuối Chương 1: Portfolio cá nhân bằng HTML

## 1. Mô tả dự án

Học viên tạo một website Portfolio cá nhân bằng HTML.  
Website chưa cần CSS đẹp, nhưng phải có cấu trúc rõ ràng, nội dung đầy đủ và dùng đúng các thẻ HTML đã học.

## 2. Cấu trúc thư mục

```text
portfolio-html
├── index.html
├── about.html
├── contact.html
└── images
    └── avatar.jpg
```

## 3. Yêu cầu bắt buộc

Website cần có:

- Cấu trúc HTML chuẩn.
- `lang="vi"`.
- `meta charset`.
- `meta viewport`.
- Tiêu đề trang.
- Header.
- Navigation.
- Main.
- Ít nhất 3 section.
- Footer.
- Ảnh đại diện có `alt`.
- Danh sách kỹ năng.
- Danh sách sở thích.
- Bảng thông tin cá nhân.
- Form liên hệ đầy đủ.
- Liên kết mạng xã hội hoặc email.
- Iframe nhúng video hoặc bản đồ.

## 4. Gợi ý bố cục

```text
Header
├── Tên website
└── Menu

Main
├── Giới thiệu
├── Ảnh đại diện
├── Kỹ năng
├── Sở thích
├── Bảng thông tin
├── Video / bản đồ
└── Form liên hệ

Footer
└── Bản quyền / thông tin liên hệ
```

| Tiêu chí | Điểm |
