# GIÁO TRÌNH CHƯƠNG 3: JAVASCRIPT CƠ BẢN

*Dành cho học viên mới học lập trình web*

> **Sản phẩm cuối chương:** Học viên nâng cấp website Portfolio cá nhân đã học ở HTML và CSS thành một website có tương tác cơ bản bằng JavaScript: nút bấm, đổi nội dung, kiểm tra form, danh sách động và mini project nhỏ.

| Thông tin | Nội dung |
| --- | --- |
| Chương | Chương 3 - JavaScript cơ bản |
| Số buổi đề xuất | 8 buổi |
| Thời lượng mỗi buổi | 2-3 giờ |
| Điều kiện trước khi học | Đã biết HTML và CSS cơ bản |
| Dự án cuối chương | Portfolio cá nhân có tương tác bằng JavaScript |

## Kế hoạch học Chương 3

| Buổi | Chủ đề | Nội dung học | Sản phẩm nhỏ |
| --- | --- | --- | --- |
| Buổi 1 | Làm quen với JavaScript | JavaScript là gì, cách gắn JS vào HTML, dùng `console.log` | Trang HTML chạy được file JavaScript riêng |
| Buổi 2 | Biến, kiểu dữ liệu và toán tử | `let`, `const`, string, number, boolean, toán tử cơ bản | Trang tính toán thông tin cá nhân đơn giản |
| Buổi 3 | Điều kiện `if else` | Kiểm tra điều kiện và đưa ra kết quả khác nhau | Chương trình kiểm tra tuổi hoặc điểm số |
| Buổi 4 | Vòng lặp và mảng | `for`, array, duyệt danh sách | Hiển thị danh sách sở thích hoặc kỹ năng |
| Buổi 5 | Hàm trong JavaScript | Tạo hàm, gọi hàm, tham số, kết quả trả về | Bộ hàm xử lý thông tin cá nhân |
| Buổi 6 | DOM cơ bản | Chọn phần tử HTML và thay đổi nội dung/giao diện | Nút đổi nội dung, đổi màu, ẩn hiện phần tử |
| Buổi 7 | Sự kiện và kiểm tra form | `click`, `input`, `submit`, kiểm tra dữ liệu nhập | Form liên hệ có kiểm tra dữ liệu |
| Buổi 8 | Hoàn thiện Portfolio có JavaScript | Tổng hợp kiến thức đã học | Portfolio cá nhân có tương tác hoàn chỉnh |

## Sản phẩm cuối chương

```text
portfolio-js
├── index.html
├── style.css
└── script.js
```

## Cách dùng bộ file

- Mỗi buổi là một file `.md` riêng.
- Có thể copy nội dung từng file vào Docusaurus, Notion, GitHub Pages hoặc tài liệu riêng.
- Nên học theo đúng thứ tự từ Buổi 1 đến Buổi 8.
