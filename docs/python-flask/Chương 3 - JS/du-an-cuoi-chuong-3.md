# Dự án cuối Chương 3: Portfolio cá nhân có JavaScript

## 1. Mô tả dự án

Học viên nâng cấp Portfolio cá nhân đã làm ở chương HTML và CSS, thêm JavaScript để website có tương tác.

Dự án không yêu cầu backend, không cần database, không cần đăng nhập.

## 2. Sản phẩm cần hoàn thành

```text
du-an-cuoi-chuong-3
├── index.html
├── style.css
└── script.js
```

## 3. Yêu cầu HTML

Website cần có:

- Header
- Phần giới thiệu
- Phần kỹ năng
- Phần sở thích hoặc dự án nhỏ
- Phần liên hệ
- Form liên hệ
- Nút bấm để JavaScript xử lý

## 4. Yêu cầu CSS

Website cần có:

- Màu nền rõ ràng
- Font chữ dễ đọc
- Card hoặc section rõ ràng
- Nút bấm được làm đẹp
- Form được định dạng
- Responsive cơ bản

## 5. Yêu cầu JavaScript

Website cần có ít nhất:

- Gắn file `script.js` đúng cách.
- Có dùng biến.
- Có dùng mảng.
- Có dùng vòng lặp.
- Có dùng hàm.
- Có chọn phần tử bằng DOM.
- Có xử lý sự kiện `click`.
- Có xử lý sự kiện `submit`.
- Có kiểm tra form.
- Có thay đổi nội dung hoặc giao diện bằng JavaScript.

## 6. Gợi ý tính năng

Học viên có thể chọn một số tính năng sau:

- Nút đổi lời giới thiệu.
- Nút đổi màu giao diện.
- Nút ẩn/hiện phần kỹ năng.
- Danh sách kỹ năng tạo từ mảng JavaScript.
- Form liên hệ kiểm tra họ tên và email.
- Thông báo gửi thành công.
- Nút quay lại đầu trang.
- Hiển thị lời chào theo tên nhập vào.

## 7. Tiêu chí đánh giá

| Tiêu chí | Điểm |
| --- | --- |
| Có đủ file HTML, CSS, JS | 1 |
| Gắn JavaScript đúng cách | 1 |
| Website có giao diện rõ ràng | 1 |
| Có dùng biến, mảng hoặc hàm | 1 |
| Có dùng DOM để chọn phần tử | 1 |
| Có xử lý sự kiện click | 1 |
| Có form liên hệ | 1 |
| Có kiểm tra form bằng JavaScript | 1 |
| Không có lỗi Console nghiêm trọng | 1 |
| Trình bày sạch, dễ hiểu | 1 |
| **Tổng điểm** | **10** |
