# CHƯƠNG 4: BACKEND VỚI FLASK VÀ POSTGRESQL

*Dành cho học viên đã học HTML, CSS và JavaScript cơ bản*

> **Sản phẩm cuối chương:** Học viên xây dựng được một website bán hàng đơn giản có backend bằng Flask, lưu dữ liệu bằng PostgreSQL, có trang sản phẩm, trang chi tiết sản phẩm, giỏ hàng và chức năng đặt hàng cơ bản.

## 2. Mục tiêu chương học

Sau khi hoàn thành Chương 4, học viên có thể:

- Hiểu backend là gì và vai trò của backend trong website.
- Biết cách tạo một ứng dụng web bằng Flask.
- Biết tổ chức thư mục Flask cơ bản.
- Biết tạo route và render HTML template.
- Biết nhận dữ liệu từ form bằng GET và POST.
- Biết kết nối Flask với PostgreSQL.
- Biết tạo model dữ liệu bằng SQLAlchemy.
- Biết thao tác CRUD với dữ liệu sản phẩm.
- Biết xây dựng trang danh sách sản phẩm và chi tiết sản phẩm.
- Biết tạo giỏ hàng cơ bản bằng session.
- Biết xử lý đặt hàng đơn giản.
- Hoàn thiện được website bán hàng có frontend, backend và database.
